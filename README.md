# KindHands

Community Giving Platform

A web application to help the Orthodox Church community collect and manage donations of money, clothes, and other items for the poor. This platform simplifies scheduling, tracking, and volunteering during donation events.

Features
Online Donations: Money, clothes, and other items.
Pickup Scheduling: Donors can select a convenient time for item collection.
Volunteer Dashboard: View pickup assignments, routes, and donation status.
Donation Tracking: Keep records of all collected items and money.
Notifications: Email/SMS reminders for donors and volunteers.
Reports & Analytics: Track total donations, active donors, and trends.
